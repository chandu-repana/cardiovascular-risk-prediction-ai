# Cardiovascular Risk Prediction AI (Multimodal)

> Built a multimodal deep learning system combining retinal imaging and clinical data for cardiovascular risk prediction with explainable AI techniques.

## 🚀 Features

* Multimodal learning (image + structured clinical data)
* CNN-based feature extraction from retinal images
* Fusion with clinical parameters for improved prediction accuracy
* Explainability using Grad-CAM
* End-to-end training and evaluation pipeline

## 🧠 Tech Stack

* Python
* TensorFlow / Keras
* CNN (Deep Learning)
* NumPy, Pandas
* Grad-CAM (Explainable AI)

## 📊 Model Highlights

* Combined visual + clinical features for better prediction
* Improved reliability via multimodal fusion
* Visual interpretability using attention maps

## 📁 Project Structure

* `src/train.py` → model training pipeline
* `src/model_builder.py` → architecture design
* `src/data_loader.py` → data handling
* `src/grad_cam.py` → explainability
* `src/evaluate_model.py` → evaluation
